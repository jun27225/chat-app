# Chat Application 💬

## 📌 Description
A real-time chat application built using Node.js that allows users to communicate instantly.

## 🚀 Features
- Real-time messaging
- Multiple users support
- Simple and responsive UI

## 🛠 Tech Stack
- Node.js
- JavaScript
- HTML
- CSS

## ▶️ How to Run
1. Install dependencies:
   npm install

2. Run the server:
   node app.js

3. Open in browser:
   http://localhost:3000
